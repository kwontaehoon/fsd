## features

애플리케이션의 핵심 비즈니스 로직과 사용자 상호작용 시나리오를 구현하는 코드 블록

- 여러 기능 + 엔티티 등이 섞인 코드 블록
- 독립적으로 사용되는 컴포넌트 모임

### 예시

- auth: 인증 관련 기능 (로그인, 로그아웃, 회원가입 등)
    - ui: 기능에 필요한 ui
    - api: entities/.../api.ts와 다르게 기능을 완성하기 위해 데이터를 가져오는 방법을 어떻게 사용할지에 대한 내용 중심
- post-creation: 게시물 생성 기능
- user-settings: 사용자 프로필 설정 변경 가능
- task-filtering: 할 일 목록 필터링 기능
- item-add-to-cart: 상품 장바구니 담기 기능
- ...
